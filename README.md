# Barid App

A modern Android application for managing disposable temporary emails. Built for speed, privacy, and simplicity.

## Features

- **Instant Email Generation**: Create disposable addresses instantly.
- **Real-time Inbox**: Auto-refresh and background notifications for new messages.
- **Always-On Service**: Customized foreground service to keep checking emails.
- **Multi-Account**: Manage multiple identities.
- **Modern UI**: Dark/Light themes, bottom navigation, and clean material design.
- **Privacy**: Local data storage with instant wipe capability.

## Technical Details

- **Architecture**: Single-Activity Model
- **Language**: Java
- **Networking**: Native OkHttp implementation
- **Storage**: SharedPreferences (JSON)

## Installation

1. Clone the repository.
2. Open in Android Studio.
3. Build and Run (Min SDK 21+).
